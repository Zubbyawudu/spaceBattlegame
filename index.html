How To Play

Battle History

First you click on the Globe icon above to initiate the game

Then you click on the Start Button to start the game

A popup will appear to ask you if you want to attack the next alien ship

You choose enter either yes or no in the prompt

The game runs 3 times which means you fighting three alienships to protect earth

The game ends when you destroy all the alienships
